# Claude + ClawX 协作协议 v2

## 对话窗口
浏览器打开 **http://localhost:9724** 实时看到双方对话

## 分工原则

**Claude（我）负责：**
- 写代码、改网站、生成内容
- 数据分析、竞品报告、策略规划
- 网页搜索、信息总结
- 任务拆分 + 给 ClawX 分配子任务

**ClawX 负责：**
- 操控桌面 App（浏览器、终端、文件管理器）
- SSH 连接服务器、执行部署命令
- 截图验证、查看界面效果
- 安装软件、配置系统

## 工作流程
1. 用户发任务 → 我分析 → 拆成两部分
2. 我干我的（写代码/分析/规划）
3. ClawX 的部分通过桥接器自动发送
4. ClawX 执行完自动回复
5. 我汇总结果 → 回报用户

## 桥接器
- 服务：`node agent-bridge.js`（在 E:\ai导航网站 运行）
- 聊天窗口：http://localhost:9724
- ClawX 自动轮询：每5秒 GET /inbox
- 无需用户手动传话
